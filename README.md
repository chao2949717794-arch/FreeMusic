# 自由音 - 免费音乐播放器

一款免费的Android音乐播放器，支持在线听歌和车载模式（Android Auto）。

## ✨ 特性

- 🎵 **免费听歌** - 无需付费，尽情享受音乐
- 🎨 **精美界面** - Material Design 3 设计，支持深浅色主题
- 🚗 **车载支持** - 完整支持Android Auto车载系统
- ❤️ **智能推荐** - 基于播放偏好的个性化推荐
- 📱 **本地存储** - 所有数据本地保存，保护隐私
- 🌙 **夜间模式** - 自适应系统主题

## 🛠️ 技术栈

- **开发语言**：Kotlin
- **UI框架**：Jetpack Compose
- **播放器**：ExoPlayer（Media3）
- **网络请求**：Retrofit + OkHttp
- **本地数据库**：Room
- **图片加载**：Coil
- **架构模式**：MVVM + Repository
